Panel - совокупность связанных визуальных элементов. Параметры модели: метод (сейчас реализованы GET и POST) для этих методов задаются URL и при необходимости параметры URL. По клику на Formation of data нужно сформировать перечень данных, которые будут приходить (с сервера) и отображаться в виде списка. При клике на create view формируется прототип панели. Вверху располагается изображение, а все остальные поля под ним. Отступы и размеры формируются в соответствии с материал дизайном. Для полей типа изображение устанавливается системное изображение (только на прототипе для наглядности, в приложении будет изображение полученное по модели), для текстовых полей выводится название поля.
